TacArc (2)
==========

fit the given parameters.

BitStream Serialization
-----------------------

| hit_something= **[bit]**
| if hit_something:
| 	if ``check_env`` parameter:
| 		**[bit]** - ???, always 0?
| 	**[u32]** - number of targets
| 		**[s64]** - target object id
| 	for each target:
| 		-> `action`
| else:
| 	if ``blocked_action`` parameter exists:
| 		**[bit]** - is blocked
| 		if blocked -> `blocked action`, else -> `miss action`
| 	else:
| 		-> miss action
